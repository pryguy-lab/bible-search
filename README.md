# Bible Verse App

> _"Your word is a lamp to my feet and a light to my path."_
> — Psalm 119:105

This web app was created entirely by AI.

It is offered freely as a tool to help people read, search, and reflect on God's Word — the Holy Bible.

---

## What It Does

- Look up any Bible verse or passage by reference (e.g. _John 3:16_, _Romans 8:28_, _Psalm 23_)
- Choose from multiple translations: **WEB**, **KJV**, **ASV**, and **BBE**
- Search Bible subjects and topics to discover relevant passages
- Save favorite verses and browse your reading history
- Customize the look and feel with themes and reading modes

---

## Tech Stack

- **Node.js** + **Express** — backend server
- **Vanilla HTML / CSS / JavaScript** — frontend (no frameworks)
- **bible-api.com** — upstream Bible text data
- **Redis** — optional caching layer
- Deployable on **Render**

---

## Running Locally

```bash
npm install
npm start
```

The app will be available at `http://localhost:3000`.
