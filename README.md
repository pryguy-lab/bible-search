## Overview

Bible Verse App is a simple web application for Bible reading and study. It allows users to search by verse reference, browse topical passages, switch translations, save favorites, and personalize the reading experience.

## Features

| Area               | What you can do                                             |
| ------------------ | ----------------------------------------------------------- |
| Verse Search       | Look up references like John 3:16, Romans 8:28, or Psalm 23 |
| Translations       | Choose WEB, KJV, ASV, or BBE                                |
| Topic Study        | Browse built-in topics and search Bible subjects            |
| Personal Study     | Save favorite passages and review history                   |
| Reading Experience | Use theme controls, focus mode, and spacing options         |

## Tech Stack

- Node.js and Express for the backend
- Vanilla HTML, CSS, and JavaScript for the frontend
- bible-api.com as the upstream Bible text API
- Redis as an optional cache layer
- Render-ready deployment configuration

## Getting Started

### Prerequisites

- Node.js 18 or newer
- npm

### Install and Run

```bash
npm install
npm start
```

App URL:

```text
http://localhost:3000
```
